NodeJS 安装 & 环境搭建

配置 npm & cnpm

没有报错就说明已经成功配置了，然后就可以下载镜像文件了

Hexo 安装 & 使用

Hexo 安装

Hexo 官网有提供的详尽的 安装使用文档，需要注意的是，Hexo 需要搭配 Git 和 NodeJS 使用，在安装之前需要把前面两个提前安装！也可以参考→BV1Yb411a7ty

Hexo 使用的是命令行进行操作，首先安装 Hexo，hexo -v 测试安装
